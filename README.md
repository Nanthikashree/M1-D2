# M1-D2
AIM:
Write a C program to read a, b value and find the smallest value using  if-else
PROGRAM:
```
#include <stdio.h>
int main()
{
    int a,b;
    scanf("%d%d",&a,&b);
    if (a<b)
    {
        printf("A is smallest.");
    }
    else
    {
        printf("B is smallest.");
    }
}
```
